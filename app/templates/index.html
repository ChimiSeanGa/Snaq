Snaq

 Snaq is a web application intended to supplement Facebook's Buy and Sell Groups. It will filter sale posts from a group the user is connected to based on the user's set preferences. The sale posts will be presented in an organized manner and properly formatted with the necessary information in order for users to easily find what they need or want.
